Wiz for linux 2.1.0版本功能介绍

 三 01 一月 2014 by eleven.i386

截图

介绍

wiz笔记 一款知识管理软件, 界面效果如上图.

在 Wiz 2.1.0 版本中 首次对Linux支持markdown语法.

( Ubuntu用户按照官网上的方法安装,不是最新版本哦, 是不支持markdown语法的 )

要求

安装Wiz 2.1.0 版本

是需要QT4/5 的支持, 另外需要cmake工具

插件

安装好之后, 点击菜单栏上的 帮助-> 关于插件

可以发现 markdown语法的支持是由于插件的原因. 因此 其他人也是可以编写自己的插件实现 不同的功能呢. 咱技术差了点. 又没有一个官方的API, 咱无法写出自己的插件. 不过有了 markdown语法支持插件 咱就足矣 :)

另外一个插件 则是网页截取. 参考为知博客

得知chrome下有一款网页截取的插件. 遂安装之, 对于长篇大论的文档. 直接点击插件,就可以 对整个页面文章区域截取,同步到你的笔记中

记.遗忘RTX密码
日 22 九月 2013 by eleven.i386

引子

在虚拟机中rtx密码是保存的,第一次拿到密码,没有做第三方的记录. 中秋期间,wine了一个rtx. 结果发现自己忘记密码了

过程

由于有了之前的遗忘密码事件 (遗忘密码之svn,记.一次邮箱密码丢失)

一开始的思路是在登录rtx的时候抓包分析,rtx的验证数据包, 不过还是习惯性的google了一下 没想到真找到了更简单的方法.

有资料显示rtx在内存中明文存储密码 by the goole

WINHEX打开RTX的内存

  1. winhex打开rtx的内存

    1. 点击图片中1的位置.open Ram
    2. 在窗口中找到rtx.
    3. 选择主要内存.Primary Memory

  2. 查找16进制字符串40000000500061007300730077006F0072006400

    1. 点击图片中1的位置.Find Hex values
    2. 在文本框内输入要搜索的16进制字符串40000000500061007300730077006F0072006400

  3. password下面的位置就是明文密码

结果

好了,明文密码得到了.只是这样的rtx是否安全呢...

wine rtx 真的很简单
